Este plugin mostra uma vitrine do Mercado de Sócio para monetizar seu blog. É importante ser cadastrado no mercado e Sócio. 

== Description ==
Este plugin é perfeito para quem deseja ganhar dinheiro com o Mercado de Sócio, ele exibe uma vitrine abaixo do seu post em seu blog/site a vitrine mostra os produtos mais relevantes do site Mercado Livre, nele é possível personalizar de acordo com seus criterios. 

Por exemplo: Exibir somente os produtos com preço minímo equivalente a R$ 100,00 e com preço maximo equivalente a R$ 5000,00. Lembrando é importante que você seja cadastrado no Mercado de Sócio para obter lucros, você precisa tero código de traqueamento do site Mercado de Sócio. Para maiores informações acesse: mercadolivre.com.br/MercadoSocios




== Installation ==

Como instalar o Plugin Mercado de Sócio! Antes de começar é necessário o cadastro no Mecado de Sócio em: mercadolivre.com.br/MercadoSocios. Obtenha seu código de traqueamento caso não saiba onde encontrar veja a vídeo aula de como pegar seu código de traqueamento. http://www.youtube.com/watch?v=SkSa2Afq_mY

e.g.

1. Envie seus arquivos para a pasta  `/wp-content/plugins/` 
2. Ative o plugin no menu de seu wordpress
3. Acesse no menu configurações ML Social e coloque seu 'código de traqueamento'. Caso não saiba  veja o vídeo como pegar seu código de traqueamento. 'http://www.youtube.com/watch?v=SkSa2Afq_mY'
